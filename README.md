# Asistente_virtual
Asistente virtual con algunas funciones básicas. 
Estos son algunos de los comandos que se pueden decir:

•	Abrir youtube
•	Abrir navegador
•	Qué día es hoy
•	Qué hora es
•	Busca en wikipedia
•	Busca en internet, 
•	Escuchar, 
•	Chiste, 
•	Enviar mensaje, 
•	Controlar pc, 
•	Tomar captura,
•	Precio de las acciones, 
•	Me voy a dormir
